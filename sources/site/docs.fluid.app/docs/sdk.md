# Fluid SDKs

Welcome to the Fluid SDK documentation. Our SDKs provide powerful tools to integrate Fluid commerce capabilities into your applications.

## Available SDKs

### FairShare SDK

The FairShare SDK provides web components for integrating FairShare commerce features into your website using a simple script tag approach. It includes:

- **Cart Widget** - Full-featured shopping cart with customizable styling
- **Lead Capture Widget** - Capture customer leads with configurable forms
- **Media Widget** - Display and interact with media content
- **JavaScript API** - Programmatic control over all commerce functions

[Get started with FairShare SDK →](https://docs.fluid.app/docs/sdk/fairshare)

### Files SDK

The Files SDK provides tools for integrating Fluid's Digital Asset Management (DAM) capabilities into your applications.
