Fluid's API gives you direct access to the customer data that powers your direct sales organization. By retrieving that data and building segments from it, you can craft marketing campaigns that speak to each customer's individual behavior and needs — whether that means reactivating lapsed buyers, recommending products based on browsing history, or tailoring promotions to a specific geography. This guide walks you through the building blocks: pulling customer data, defining segments, and using those segments to drive action.

## Step 1: Access Your Customer Data

Use the `/customers` endpoint to retrieve your company's customer list. Pass your **company token** as a Bearer token in the `Authorization` header.

```javascript
const url = `https://api.fluid.app/v1/companies/${companyId}/customers`;

fetch(url, {
  headers: {
    'Authorization': `Bearer ${companyToken}`,
  },
})
  .then(response => response.json())
  .then(data => {
    const customers = data.data;
    // Pass customers to your segmentation logic below
  })
  .catch(error => console.error(error));
```

<Note>
  Replace `companyId` with your company's ID and `companyToken` with a valid company token. See the [Authentication guide](/guides/authentication) for token details.
</Note>

## Step 2: Build Customer Segments

Once you have the customer list, apply filters to group customers into segments that match your campaign goals.

### Purchase History

Identify customers who have not purchased within a defined window to build a reactivation audience.

```javascript
const SIX_MONTHS_AGO = new Date();
SIX_MONTHS_AGO.setMonth(SIX_MONTHS_AGO.getMonth() - 6);

const inactiveCustomers = customers.filter(customer => {
  const lastPurchase = new Date(customer.last_purchase_at);
  return lastPurchase < SIX_MONTHS_AGO;
});
```

### Browsing Behavior

Segment users based on products or categories they have viewed to surface relevant recommendations.

```javascript
const viewedProductId = 'PROD_456';

const interestedCustomers = customers.filter(customer =>
  customer.viewed_product_ids?.includes(viewedProductId)
);
```

### Demographics

Tailor promotions to customers in a specific location or age group.

```javascript
const localCustomers = customers.filter(customer =>
  customer.state === 'CA'
);
```

## Step 3: Act on Your Segments

Use the unique identifiers from each segment — typically customer IDs — to drive targeted outreach.

### Abandoned Cart Reminders

Loop through customers who added items to their cart but never checked out, and trigger a reminder through your email or messaging provider.

```javascript
// Identify customers with abandoned carts (replace with your cart-detection logic)
const abandonedCarts = customers.filter(customer => customer.has_abandoned_cart);

abandonedCarts.forEach(customer => {
  sendAbandonedCartEmail({
    to: customer.email,
    firstName: customer.first_name,
    cartItems: customer.cart_items,
  });
});
```

<Tip>
  Combine purchase history and cart data for more precise targeting — for example, send a steeper discount to customers who abandoned a cart *and* haven't purchased in over 90 days.
</Tip>
