The Fluid Mobile SDK enables native iOS and Android applications to integrate directly with Fluid commerce features. Rather than relying on web views or hybrid approaches, the Mobile SDK will provide first-class native APIs that feel at home on each platform — giving your users a seamless, high-performance commerce experience inside your app.

<Note>
The Mobile SDK documentation is actively being developed. Check back soon for comprehensive guides, API references, and code samples. In the meantime, explore the related resources below for integration options that are available today.
</Note>
